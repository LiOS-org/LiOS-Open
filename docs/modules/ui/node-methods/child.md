# Node Methods : New child
The `child` method is used to create a new child element and append it to the current element. It takes a string as an argument which specifies the tag name of the new child element.

```JS
const newChild = main.child("div");
```

The `child` method returns the newly created child element, which also has access to all the [node methods](../node-methods.md).
