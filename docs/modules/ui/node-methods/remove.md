# Node Method: remove

This is a handy method to remove the element from the DOM. It is useful when you want to remove an element after a certain event or condition is met.

```JS
newChild.remove(); // removes the newChild element from the DOM
```
> Note: The remove method automatically removes all event listeners attached to the element, so you don't have to worry about memory leaks when using this method.
