# Node Methods: Text

The `text` method is used to set the text content of the element. It takes a string as an argument and sets it as the text content of the element.

```JS
main.text("Hello World!");
```

> Note: The `text` method will replace any existing text content of the element, if you want to append text to the existing content, you can use the [`child()`](child.md) method to create a new child element and set its text content.
