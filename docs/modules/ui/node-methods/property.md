# Node Methods: CSS Properties

The `property` method is used to set the CSS attributes of the element. It takes an object as an argument where the keys are the attribute names and the values are the corresponding values for those properties.

```JS
main.property({
    "--lios-button-background": "Green",
    "--lios-button-hover-background": "LightGreen"
});
```
