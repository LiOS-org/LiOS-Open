# Node Methods : Class

The `class` method is used to add or remove classes from the element. It takes a string as an argument and adds it as a class to the element. 

## class.add(value)

```JS
main.class.add("main-class");

//  This method can also take multiple values and will add them all

main.class.add("class1", "class2", "class3");
```

## class.remove(value)

```JS
main.class.remove("main-class");
//  This method can also take multiple values and will remove them all
main.class.remove("class1", "class2", "class3");
```
