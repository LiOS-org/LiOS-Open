# Node Methods : ID

The `id` method is used to set the id of the element. It takes a string as an argument and sets it as the id of the element.

The `id` should be unique to the element, it is used to identify the element in the DOM and can be used to select the element using CSS selectors or JavaScript.

```JS
main.id("main-element");
```
