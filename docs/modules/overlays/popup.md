# Pop Ups

Pop-ups are UI elements that appear on top of the screen. Their main goal is to quickly attract the user's attention. Pop-ups can be used for alerts, reminders, and many other things.

This submodule currently provides a generic popup which you can customize to your needs.


## Creating a new Pop-Up

```JS
const myPopup = overlay.popup();
```

- This will create a blank generic popup.
- For this to work, you will need to have a `<body>` element in the `DOM`.


## Populating the pop-up

- Once a pop-up is created, it exposes several methods, including [node methods](../ui/node-methods.md), which can be used to populate the pop-up.
  
## Methods

1. myPopup.open()
   
   This method will Open the popup

2. myPopup.close()
   
   This method will Close the popup

3. myPopup.delete()
   
   This method will delete the popup from both the `DOM` tree and the `vDOM` and also it will remove all the event listeners

4. myPopup.background(value)
   
   Will change the background to the provided value

> Note: Popup will recieve an autogenerated `id` and the node method `id()` has been remove to make it hard to reassign `id`, this was done intentionally to keep every popup distinct
