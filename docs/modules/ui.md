# UI module

This is a pure `JS` module that provides a unified interface for DOM manipulation and event handling. It serves as the foundation for other modules, enabling seamless integration and consistent behavior across the entire system.

## Getting Started
To get started with the `UI module`, you need to install it first. You can do this by importing the module into your project and initializing it.

```JS
import { liosOpen } from "${path to liosOpen installation}/liosOpen.js";
```

```JS
const ui = liosOpen.ui;
```
