# Effects

This module provides pre built CSS effects for the user interface.

To get started, first of all install the `CSS` file into your project
```CSS
@import url(${path-to-lios-open}/modules/css/effects/effects.css);
```

This module supports two types of usage

1. Standalone `CSS` usage: Use it directly in HTML and CSS.
2. As an UI extension: Use it with `JS` via the `UI` module
